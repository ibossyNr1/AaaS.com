TikTok Chromatic Text Effect Recipe

To recreate the TikTok/CRT text effect in Canva:

1. Place the main text in white (#FFFFFF)
2. Duplicate the text layer — change color to #00FFC2 (green), offset -2px left, -1px up, set opacity to 60%, blend mode Screen
3. Duplicate again — change color to #F43F6C (red), offset +2px right, +1px down, set opacity to 60%, blend mode Screen
